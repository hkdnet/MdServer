MdServer
====

markdownで書きためたものを公開するアプリケーションです。

## Description

### 機能

* markdownを閲覧できます
* トピックごとにフォルダを区切っても使えます
* シンプルなので好きに拡張できます。

### 不得意分野とか課題とか

* 自分でサーバをたてる必要があります
* パフォーマンス考えてません
  * 小さなチームで利用する程度だったら特に問題ないと思いますが……
* セキュリティ考えてません

### 今後の予定とか

* cssも置かれたのを適用したい
* 一覧のフォルダ表示をもうすこしかっこよくしたい
* フォルダにアクセスしたらその中の一覧を出したい
* フォルダごとにcss適用したりしたい

## 対抗馬

いっぱいありそうなのでイケてるやつあったらおしえてください

## 必要要件

* Ruby

## 使い方

* `$ git clone hogehoge`
* `$ cd MdServer`
* `$ rackup`
* accsess to `localhost:9292`
